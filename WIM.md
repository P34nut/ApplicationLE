Molehill Empire 2 started as the mobile version of the browser game 'Molehill Empire' from 2006. 
By now it also has a web, standalone and steam version. 
It is a full 2D game with lots of UIs. The main gameplay loop is to plant, harvest and sell different vegetables and fruits.
The crop can also be used in different features to gain extra Xp or currency. 

When I started at upjers this was my first project I worked on. It was already softlaunched but I became the sole developer after about 3 months. 
I coordinated the global release, added new features. I also made the game ready for a web and steam release, which inclueded adding a landscape mode that was not planed when the development first began. 
The game is still getting new features every now and then.

More infos about the game and the web version can be found [here](https://en.upjers.com/molehill-empire-2)
